Python-MNE全套教程（官网翻译）-从不同的系统中读取数据01：从MEN设备导入

MEGIN/Elekta Neuromag VectorView and TRIUX (.fif)

用mne.io.read_raw_fif()加载。如果数据是在打开MaxShield的情况下记录的，并且没有用MaxFilter处理，需要用mne.io.read_raw_fif(…allow_maxshield = True)。

FIL OPM (.bin)

用mnei .io.read_raw_fil()

Artemis123 (.bin)

用mne.io.read_raw_artemis123()

4-D Neuroimaging / BTI data (dir)

用mne.io.read_raw_bti()

CTF data (dir)

用mne.io.read_raw_ctf()

CTF Polhemus data

用mne.channels.read_dig_polhemus_isotrak()

应用软件梯度补偿

这是用CTF系统的操作细节，详细查看应用梯度补偿

Ricoh/KIT MEG system data (.con/.sqd)

用 mne.io.read_raw_kit() 和 mne.read_epochs_kit()

FieldTrip MEG/EEG data (.mat)

用mne.io.read_raw_fieldtrip()， mne.read_epochs_fieldtrip()和mne.read_evoked_fieldtrip()来读取来自FieldTrip的数据。

Creating MNE data structures from arbitrary data (from memory)

通过使用mne.io，可以从内存中构造任意数据。
